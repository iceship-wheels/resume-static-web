## 技术选型
| 项目 | 选择 | 理由 |
|------|------|------|
| 框架 | Astro | 零 JS 默认输出、轻量、内容优先 |
| 样式 | Tailwind CSS | 原子化、快速迭代、体积可控 |
| 交互 | Alpine.js | 轻量（~15KB）、声明式、够用 |
| 部署 | GitHub Pages | 免费、与仓库集成、CI/CD 友好 |
| 多媒体 | 原生 HTML5 + CSS | 视频/图片无需额外依赖 |
